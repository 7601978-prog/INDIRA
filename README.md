# Пульс отдела

Статический прототип приложения для ведения рутинных задач, проектов, инициатив и нагрузки команды.

## Как открыть

Откройте `index.html` в браузере.

## Что есть в прототипе

- отдельные разделы для рутины и проектов/инициатив;
- добавление задач с ответственным, статусом, приоритетом, сроком и часами;
- переключение статусов и удаление записей;
- расчет загрузки каждого члена команды;
- CSV-выгрузка задач, проектов и нагрузки;
- локальное сохранение данных в браузере.
